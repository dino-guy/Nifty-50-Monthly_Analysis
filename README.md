## 🎯 Project Objective

This project focuses on technical analysis of selected Nifty 50 stocks using price action concepts. The objective was to identify trends, support and resistance levels, and create trade setups based on risk-reward analysis.

---

## 🛠️ Tools Used

| Tool            | Purpose                              |
| --------------- | ------------------------------------ |
| TradingView     | Chart Analysis                       |
| Groww           | Market Tracking                      |
| Microsoft Excel | Trade Journal & Performance Tracking |
| GitHub          | Project Documentation                |

---

## 📊 Methodology

### 1️⃣ Trend Analysis

* Identified Uptrend 📈
* Identified Downtrend 📉
* Identified Sideways Market ↔️
* Analyzed price action using TradingView charts.

### 2️⃣ Support & Resistance

* Marked key support levels based on historical price reactions.
* Identified major resistance zones.
* Tracked market reaction around important price levels.

### 3️⃣ Trade Setup Creation

For each stock:

* Entry Price
* Stop Loss
* Target Price
* Risk-Reward Ratio

### 4️⃣ Smart Money Concepts (SMC)

* BOS (Break of Structure)
* MSS (Market Structure Shift)
* Liquidity Analysis
* Risk-Reward Planning

### 5️⃣ Trade Documentation

* Recorded all trade setups in Excel.
* Maintained a structured trading journal.
* Tracked trade outcomes and observations.

### 6️⃣ Performance Review

* Evaluated trade outcomes.
* Reviewed risk and reward metrics.
* Identified strengths and improvement areas.
